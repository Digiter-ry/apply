1. Copy this folder to `C:\xampp\htdocs\apply\jobs` (Windows) or `/opt/lampp/htdocs/apply/jobs` (Linux).
2. Start Apache (XAMPP Control Panel).
3. Open: http://localhost/apply/jobs/
